# Story: STORY-008 Build global setup page for vehicles

## Metadata
- **Priority**: P1
- **Status**: planned

## User story
As a planner, I want a global setup page where I can manage vehicles, so that my trip planning uses reusable vehicle profiles.

## Mode ownership
- Primary mode: SETUP
- Secondary modes: PLAN

## Acceptance criteria
- [ ] Setup page is accessible from app navigation.
- [ ] Vehicle list displays all configured vehicles.
- [ ] User can open create/edit forms from this page.

## Dependencies
- Vehicle data model and APIs
- App navigation/settings shell
