# Research Note: RES-001 Map provider evaluation for EPIC-001

## Question
Which map provider/library should be used for the initial web map implementation?

## Method
- Compare existing stack compatibility.
- Evaluate licensing, rendering performance, and developer ergonomics.

## Findings
- Decision pending implementation context.
- For EPIC-001 scope, map can start with lightweight integration and evolve.

## Recommendation
Choose the provider already used elsewhere in the product stack (if any). If none exists, prioritize one with straightforward web integration and clear licensing terms.
